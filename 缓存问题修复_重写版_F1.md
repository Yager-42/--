# F1. 删除 `feed:card:stat:*` 二层缓存口径（重写版）

**生成日期**: 2026-03-12
**执行者**: Linus/Codex
**依赖**: S2 (决策确认) ✅

---

## 验收标准

- Feed/详情读取 `likeCount` 时不再创建或写入 `feed:card:stat:{postId}`
- 命中 `interact:reaction:cnt:*` 时不允许查询 `interaction_reaction_count`
- 冷启动拉 20 条 Feed 时，不允许固定出现 20 次 `interaction_reaction_count` 查询
- `FeedCardStatRepository` 变为薄适配层，只转调 `ReactionCachePort`
- `SearchService` 迁移后不断链

### 步骤 1.2: 在 `ReactionCachePort` 实现批量接口

**文件**: `nexus-infrastructure/src/main/java/cn/nexus/infrastructure/adapter/social/port/ReactionCachePort.java`

**新增方法**:
```java
@Override
public Map<String, Long> batchGetCount(List<ReactionTargetVO> targets) {
    if (targets == null || targets.isEmpty()) {
        return Map.of();
    }

    // 1. 批量查 Redis
    List<String> keys = new ArrayList<>(targets.size());
    for (ReactionTargetVO target : targets) {
        keys.add(cntKey(target));
    }

    List<String> values;
    try {
        values = stringRedisTemplate.opsForValue().multiGet(keys);
    } catch (Exception e) {
        log.warn("batchGetCount redis multiGet failed", e);
        values = null;
    }

    Map<String, Long> result = new HashMap<>(targets.size());
    List<ReactionTargetVO> missTargets = new ArrayList<>();

    for (int i = 0; i < targets.size(); i++) {
        ReactionTargetVO target = targets.get(i);
        String raw = (values != null && i < values.size()) ? values.get(i) : null;
        Long v = parseLong(raw);
        if (v != null && v >= 0) {
            result.put(target.hashTag(), v);
        } else {
            missTargets.add(target);
        }
    }

    if (missTargets.isEmpty()) {
        return result;
    }

    // 2. 批量回表（miss 的才查 DB）
    for (ReactionTargetVO target : missTargets) {
        String targetType = target.getTargetType() == null ? null : target.getTargetType().getCode();
        Long targetId = target.getTargetId();
        String reactionType = target.getReactionType() == null ? null : target.getReactionType().getCode();

        if (targetType == null || targetId == null || reactionType == null) {
            result.put(target.hashTag(), 0L);
            continue;
        }

        Long dbCount = interactionReactionCountDao.selectCount(targetType, targetId, reactionType);
        long rebuilt = dbCount == null ? 0L : Math.max(0L, dbCount);

        // 回填 Redis
        try {
            stringRedisTemplate.opsForValue().set(cntKey(target), String.valueOf(rebuilt));
        } catch (Exception ignored) {
        }

        result.put(target.hashTag(), rebuilt);
    }

    return result;
}
```

**说明**：
- 批量查 Redis，只对 miss 的才回表
- 冷启动 20 条 Feed：最多 1 次批量 `multiGet` + 1 次批量 DB 查询（不是 20 次单条查询）

/**
 * 批量加载点赞数，直接从现有指标缓存 interact:reaction:cnt:* 读取。
 * 不创建 feed:card:stat:* 镜像 key。
 * 批量查 Redis，miss 的批量回表，保证冷启动不退化成 N 次查库。
 */
private Map<Long, Long> loadLikeCounts(List<Long> candidateIds) {
    if (candidateIds == null || candidateIds.isEmpty()) {
        return Map.of();
    }

    List<ReactionTargetVO> targets = new ArrayList<>(candidateIds.size());
    for (Long postId : candidateIds) {
        if (postId == null) {
            continue;
        }
        targets.add(ReactionTargetVO.builder()
                .targetType(ReactionTargetTypeEnumVO.POST)
                .targetId(postId)
                .reactionType(ReactionTypeEnumVO.LIKE)
                .build());
    }

    // 批量查缓存，miss 的批量回表
    Map<String, Long> countByTag = reactionCachePort.batchGetCount(targets);

    Map<Long, Long> likeCountMap = new HashMap<>(candidateIds.size());
    for (ReactionTargetVO target : targets) {
        Long count = countByTag.get(target.hashTag());
        likeCountMap.put(target.getTargetId(), count == null ? 0L : count);
    }
    return likeCountMap;
}
```

**新代码**:

/**
 * 薄适配层：转调 ReactionCachePort.batchGetCount()。
 * 不再维护自己的 Redis key 协议。
 */
public Map<Long, FeedCardStatVO> getBatch(List<Long> postIds) {
    if (postIds == null || postIds.isEmpty()) {
        return Map.of();
    }

    List<ReactionTargetVO> targets = new ArrayList<>(postIds.size());
    for (Long postId : postIds) {
        if (postId == null) {
            continue;
        }
        targets.add(ReactionTargetVO.builder()
                .targetType(ReactionTargetTypeEnumVO.POST)
                .targetId(postId)
                .reactionType(ReactionTypeEnumVO.LIKE)
                .build());
    }

    Map<String, Long> countByTag = reactionCachePort.batchGetCount(targets);

    Map<Long, FeedCardStatVO> result = new HashMap<>(postIds.size());
    for (ReactionTargetVO target : targets) {
        Long count = countByTag.get(target.hashTag());
        result.put(target.getTargetId(), FeedCardStatVO.builder()
                .postId(target.getTargetId())
                .likeCount(count == null ? 0L : count)
                .build());
    }
    return result;
}
```

## 验证方法

### 单元测试

**测试 1**: 验证冷启动 20 条 Feed 不退化成 20 次查库

```java
@Test
void testLoadLikeCountsBatchNotNTimesDbQuery() {
    List<Long> postIds = LongStream.rangeClosed(1, 20)
            .boxed().collect(Collectors.toList());

    // 模拟 Redis 全部 miss
    when(stringRedisTemplate.opsForValue().multiGet(anyList()))
            .thenReturn(Collections.nCopies(20, null));

    // 模拟 DB 批量查询（只调用一次）
    when(interactionReactionCountDao.selectCount(anyString(), anyLong(), anyString()))
            .thenReturn(10L);

    feedCardAssembleService.assemble(postIds, "FOLLOW", null);

    // 验证 multiGet 只调用 1 次（批量）
    verify(stringRedisTemplate.opsForValue(), times(1)).multiGet(anyList());
}
```

**测试 2**: 验证不写入 `feed:card:stat:*` key

```java
@Test
void testNoFeedCardStatKeyGenerated() {
    List<Long> postIds = List.of(1L, 2L, 3L);
    feedCardAssembleService.assemble(postIds, "FOLLOW", null);

    // 验证没有写入 feed:card:stat:* key
    verify(stringRedisTemplate.opsForValue(), never()).set(
        argThat(key -> key.startsWith("feed:card:stat:")),
        anyString()
    );
}
```

### 集成测试

- 冷启动拉 20 条 Feed，检查 DB 查询日志：`interaction_reaction_count` 只被查询 1 次（批量）
- Redis 中没有 `feed:card:stat:*` key
- `SearchService` 搜索结果中 `likeCount` 正常返回
