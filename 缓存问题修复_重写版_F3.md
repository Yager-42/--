# F3. 统一 `ContentRepository.findPost` 协议（重写版）

**生成日期**: 2026-03-12
**执行者**: Linus/Codex
**依赖**: S2 (决策确认) ✅

---

## 验收标准

- `findPost` 走 Redis L2 + single-flight，不再旁路直接打 DB
- 命中 `"NULL"` 不回源
- 并发 miss 只允许一次回源
- 保持 `selectById` 的原 SQL 语义（查所有状态，无过滤）
- 只有 `findPost` 有资格写 NULL（因为它走 `selectById` 真相源）
- `listPostsByIds` 读到缓存后必须过滤 `status=2`，不能直接全收
- `listPostsByIds` 对 miss 的 id **禁止写 NULL**（因为 miss 可能只是"不是已发布状态"）

---

## 核心协议（必须严格遵守）

### 协议 1：`interact:content:post:*` 的真义

**定义**：存储 `selectById` 查出来的"帖子真相对象"（所有状态，无过滤）

**写入资格**：
- ✅ `findPost()` 可以写（走 `selectById`）
- ❌ `listPostsByIds()` 不能写 NULL（走 `selectByIds` 只查 status=2）

**读取后处理**：
- `findPost()` 读到缓存后直接返回（无需过滤）
- `listPostsByIds()` 读到缓存后必须过滤 `status=2`

### 协议 2：NULL 哨兵的写入资格

**只有以下情况才能写 `"NULL"`**：
- 走 `selectById` 真相源
- 确认 DB 真没有记录（不是"查不到已发布"，而是"真不存在"）

**禁止写 NULL 的情况**：
- `listPostsByIds()` 的 miss（可能只是 status != 2）
- 任何只查"已发布"的场景的 miss

**新代码**:
```java
@Override
@Transactional(readOnly = true)
public ContentPostEntity findPost(Long postId) {
    if (postId == null) {
        return null;
    }

    // L1 热点缓存（保留原逻辑）
    boolean hot = isHotKeySafe(hotkeyKey(postId));
    if (hot) {
        ContentPostEntity cached = postCache.getIfPresent(postId);
        if (cached != null) {
            return copyPost(cached);
        }
    }

    // L2 Redis 缓存（新增）
    String redisKey = postRedisKey(postId);
    String cached = null;
    try {
        cached = stringRedisTemplate.opsForValue().get(redisKey);
    } catch (Exception e) {
        log.warn("findPost redis get failed, key={}", redisKey, e);
    }

    // 命中 NULL 哨兵，直接返回 null
    if (POST_REDIS_NULL_VALUE.equals(cached)) {
        return null;
    }

    // 命中正向缓存，解析并返回
    if (cached != null) {
        if (cached.isBlank()) {
            // 坏 key：删除后按 miss 处理
            deleteRedisQuietly(redisKey);
        } else {
            ContentPostEntity entity = parsePostCache(cached);
            if (entity != null && entity.getPostId() != null) {
                // 写入 L1（如果是热点）
                if (hot) {
                    postCache.put(postId, copyPost(entity));
                }
                return copyPost(entity);
            }
            // 解析失败：删除坏 key 后按 miss 处理
            deleteRedisQuietly(redisKey);
        }
    }

    // single-flight 回源（新增）
    ContentPostEntity post = singleFlight.execute(String.valueOf(postId), () -> {
        // leader 内二次查 Redis（跨节点协调）
        String secondCheck = null;
        try {
            secondCheck = stringRedisTemplate.opsForValue().get(redisKey);
        } catch (Exception ignored) {
        }

        if (POST_REDIS_NULL_VALUE.equals(secondCheck)) {
            return null;
        }

        if (secondCheck != null && !secondCheck.isBlank()) {
            ContentPostEntity entity = parsePostCache(secondCheck);
            if (entity != null && entity.getPostId() != null) {
                return entity;
            }
            deleteRedisQuietly(redisKey);
        }

        // 回源：保持原 selectById 语义（查所有状态）
        ContentPostEntity rebuilt = toPostEntity(contentPostDao.selectById(postId));
        fillPostTypes(rebuilt == null ? List.of() : List.of(rebuilt));
        fillPostContent(rebuilt == null ? List.of() : List.of(rebuilt));

        // 写回 Redis
        if (rebuilt == null) {
            // 只有 selectById 确认不存在，才写 NULL
            try {
                stringRedisTemplate.opsForValue().set(
                    redisKey,
                    POST_REDIS_NULL_VALUE,
                    nullTtlSeconds(),
                    TimeUnit.SECONDS
                );
            } catch (Exception ignored) {
            }
        } else {
            // 写正向缓存
            try {
                String json = serializePostCache(rebuilt);
                stringRedisTemplate.opsForValue().set(
                    redisKey,
                    json,
                    positiveTtlSeconds(),
                    TimeUnit.SECONDS
                );
            } catch (Exception e) {
                log.warn("findPost redis set failed, key={}", redisKey, e);
            }
        }

        return rebuilt;
    });

    // 写入 L1（如果是热点且非空）
    if (hot && post != null) {
        postCache.put(postId, copyPost(post));
    }

    return post == null ? null : copyPost(post);
}
```

**关键点**：
- 保持 `selectById` 的原 SQL（查所有状态）
- 不复用 `listPostsByIds`
- 只加缓存层，不改语义
- 只有 `findPost` 有资格写 NULL

### 步骤 3.4: 补充辅助方法

**文件**: `nexus-infrastructure/src/main/java/cn/nexus/infrastructure/adapter/social/repository/ContentRepository.java`

**位置**: 在 `parsePostCache` 方法附近新增

**新增方法**:

private String serializePostCache(ContentPostEntity post) {
    if (post == null) {
        return null;
    }
    try {
        return objectMapper.writeValueAsString(post);
    } catch (Exception e) {
        log.warn("serialize post cache failed, postId={}", post.getPostId(), e);
        return null;
    }
}

private long positiveTtlSeconds() {
    // 60~75s 抖动
    return 60 + ThreadLocalRandom.current().nextLong(16);
}

private long nullTtlSeconds() {
    // 30~40s 抖动
    return 30 + ThreadLocalRandom.current().nextLong(11);
}
```

@Test
void testFindPostConcurrentMissOnlyOneDbQuery() throws Exception {
    Long postId = 123L;
    int concurrency = 10;
    CountDownLatch latch = new CountDownLatch(concurrency);
    AtomicInteger dbQueryCount = new AtomicInteger(0);

    when(contentPostDao.selectById(postId)).thenAnswer(inv -> {
        dbQueryCount.incrementAndGet();
        Thread.sleep(100);
        return mockPublishedPost(postId);
    });

    ExecutorService executor = Executors.newFixedThreadPool(concurrency);
    for (int i = 0; i < concurrency; i++) {
        executor.submit(() -> {
            try { contentRepository.findPost(postId); }
            finally { latch.countDown(); }
        });
    }
    latch.await();

    // 验证只查询 DB 1 次
    assertEquals(1, dbQueryCount.get());
}
```
