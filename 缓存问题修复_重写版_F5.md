# F5. 详情页改为复用 post key + 本地 L1（重写版）

**生成日期**: 2026-03-12
**执行者**: Linus/Codex
**依赖**: F3 ✅（findPost 已走 interact:content:post:* 缓存）

---

## 验收标准

- `ContentDetailQueryService` 不再维护 `interact:content:detail:*` Redis key
- 详情页复用 `ContentRepository.findPost()` 走 `interact:content:post:*`
- 保留本地 L1（Caffeine，TTL 1 小时）
- 正文复用 `post.getContentText()`，不再命中缓存后又查一次 KV
- `grep` 业务代码时，`interact:content:detail:` 的 Redis 读写残留为 0

---

## 核心变化

**删除**：
- `REDIS_KEY_PREFIX = "interact:content:detail:"`
- `rebuildSnapshot()` 中的 Redis 读写逻辑（`cacheNull`、`cacheValue`、`redisKey`、`stableSnapshotTtl`）
- `REDIS_TTL_SECONDS`、`REDIS_TTL_JITTER_SECONDS`、`REDIS_NULL_TTL` 常量

**保留**：
- 本地 L1（`localCache`，Caffeine，TTL 1 小时）
- `singleFlight`（进程内并发去重）
- `evictLocal()` 方法（供 `ContentCacheEvictPort` 调用）

**改变**：
- `query()` 的回源路径：从"查 Redis detail key → 回源 DB+KV"改为"调 `contentRepository.findPost()` → 复用 post key 缓存"
- 正文来源：从"每次查 KV"改为"复用 `post.getContentText()`"

---

## 实施步骤

### 步骤 5.1: 修改 `ContentDetailQueryService.query()`

**文件**: `nexus-trigger/src/main/java/cn/nexus/trigger/http/social/support/ContentDetailQueryService.java`

**旧代码**（第 61-97 行）:
```java
public ContentDetailResponseDTO query(Long postId) {
    if (postId == null) {
        throw new AppException(ResponseCode.ILLEGAL_PARAMETER.getCode(), "postId is required");
    }

    StableSnapshot local = localCache.getIfPresent(postId);
    if (local != null) {
        return buildResponse(local);
    }

    String redisKey = redisKey(postId);
    String cached = null;
    try {
        cached = stringRedisTemplate.opsForValue().get(redisKey);
    } catch (Exception e) {
        log.warn("content detail redis get failed, key={}", redisKey, e);
    }
    if (REDIS_NULL.equals(cached)) {
        throw new AppException(ResponseCode.NOT_FOUND.getCode(), ResponseCode.NOT_FOUND.getInfo());
    }
    if (cached != null) {
        if (cached.isBlank()) {
            deleteRedisQuietly(redisKey);
        } else {
            StableSnapshot snapshot = parse(cached);
            if (snapshot != null) {
                localCache.put(postId, snapshot);
                return buildResponse(snapshot);
            }
            deleteRedisQuietly(redisKey);
        }
    }

    StableSnapshot snapshot = singleFlight.execute(String.valueOf(postId), () -> rebuildSnapshot(postId, redisKey));
    localCache.put(postId, snapshot);
    return buildResponse(snapshot);
}
```

**新代码**:
```java
public ContentDetailResponseDTO query(Long postId) {
    if (postId == null) {
        throw new AppException(ResponseCode.ILLEGAL_PARAMETER.getCode(), "postId is required");
    }

    // L1 本地缓存（保留）
    StableSnapshot local = localCache.getIfPresent(postId);
    if (local != null) {
        return buildResponse(local);
    }

    // single-flight 进程内并发去重（保留）
    StableSnapshot snapshot = singleFlight.execute(String.valueOf(postId), () -> {
        // leader 内再查一次 L1（避免并发等待者重复构建）
        StableSnapshot secondCheck = localCache.getIfPresent(postId);
        if (secondCheck != null) {
            return secondCheck;
        }

        // 复用 ContentRepository.findPost()，走 interact:content:post:* 缓存
        ContentPostEntity post = contentRepository.findPost(postId);
        if (post == null || post.getPostId() == null) {
            throw new AppException(ResponseCode.NOT_FOUND.getCode(), ResponseCode.NOT_FOUND.getInfo());
        }

        // 构建稳定快照
        // post.getContentText() 已由 findPost() 调用 fillPostContent() 从 KV 填充
        // 不需要再查一次 KV
        return StableSnapshot.builder()
                .postId(post.getPostId())
                .authorId(post.getUserId())
                .title(post.getTitle())
                .content(post.getContentText() == null ? "" : post.getContentText())
                .summary(post.getSummary())
                .summaryStatus(post.getSummaryStatus())
                .mediaType(post.getMediaType())
                .mediaInfo(post.getMediaInfo())
                .locationInfo(post.getLocationInfo())
                .status(post.getStatus())
                .visibility(post.getVisibility())
                .versionNum(post.getVersionNum())
                .edited(post.getEdited())
                .createTime(post.getCreateTime())
                .build();
    });

    // 写入 L1
    localCache.put(postId, snapshot);
    return buildResponse(snapshot);
}
```

### 单元测试

**测试 1**: 验证 `query()` 不再读写 `interact:content:detail:*`

```java

void testQueryNoDetailRedisKey() {
    Long postId = 123L;
    ContentPostEntity mockPost = mockPublishedPost(postId);

    when(contentRepository.findPost(postId)).thenReturn(mockPost);

    contentDetailQueryService.query(postId);

    // 验证没有读写 interact:content:detail:* key
    verify(stringRedisTemplate, never()).opsForValue();
}
```
